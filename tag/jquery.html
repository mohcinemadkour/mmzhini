Calendario, un calendrier design et responsive jQuery

Vous cherchez un moyen simple de réaliser un calendrier sur votre site web ? alors Calendario est fait pour vous.

Calendario est un plug-in jQuery permettant la création dynamique de calendrier sympa et compatible responsive webdesing.

Tous les évènements peuvent être importés à partir d'un fichier JSON. Mais attention il n'est malheureusement pas compatible avec tous les navigateurs, car il utilise la méthode calc().
